1. Основы информатики

Информатика — это наука о обработке информации. Начни с понимания базовых понятий:

  • Биты и байты: Информация хранится в двоичном виде (0 и 1). 1 байт = 8 бит. Например, текст "A" в ASCII — 01000001.
  • Алфавит и кодировка: Текст кодируется, например, в UTF-8 для поддержки русского языка.
  • Логические операции: AND (И), OR (ИЛИ), NOT (НЕ). Они используются в процессорах для вычислений.

Пример логической таблицы для AND:

1.2. Структуры данных

Массивы, списки, стеки, очереди. Массив — фиксированный набор элементов одного типа. Список — динамический.

Пример массива: [1, 2, 3]. Операции: доступ по индексу O(1), вставка O(n).

1.3. Компьютерная архитектура

Компьютер состоит из CPU (процессор), RAM (память), HDD/SSD (хранение). Процессор выполняет инструкции по модели фон Неймана.

2. Алгоритмы

Алгоритм — это последовательность шагов для решения задачи. Они оцениваются по времени (O-нотация) и памяти.

2.1. Алгоритмы сортировки

Сортировка упорядочивает элементы. Вот популярные:

Сортировка пузырьком (Bubble Sort)

Сравнивает соседние элементы.

Для i от 0 до n-1:
    Для j от 0 до n-i-1:
        Если a[j] > a[j+1]:
            Поменять a[j] и a[j+1]

Пример: [5, 3, 8, 4] → [3, 5, 4, 8] после первого прохода.

Сортировка вставками (Insertion Sort)

Для i от 1 до n-1:
    key = a[i]
    j = i - 1
    Пока j >= 0 и a[j] > key:
        a[j+1] = a[j]
        j = j - 1
    a[j+1] = key

Сортировка выбором (Selection Sort)

Для i от 0 до n-1:
    min_idx = i
    Для j от i+1 до n:
        Если a[j] < a[min_idx]:
            min_idx = j
    Поменять a[i] и a[min_idx]

Быстрая сортировка (Quick Sort)

Функция quicksort(a, low, high):
    Если low < high:
        pivot_idx = partition(a, low, high)
        quicksort(a, low, pivot_idx - 1)
        quicksort(a, pivot_idx + 1, high)

Сортировка слиянием (Merge Sort)

Функция merge_sort(a):
    Если len(a) <= 1: Вернуть a
    mid = len(a) // 2
    left = merge_sort(a[:mid])
    right = merge_sort(a[mid:])
    Вернуть merge(left, right)

2.2. Алгоритмы поиска

Поиск находит элемент в данных.

Линейный поиск (Linear Search)

Проверяет по порядку. O(n).

Для i от 0 до n-1:
    Если a[i] == target:
        Вернуть i
Вернуть -1

Пример: В [1, 3, 5] найти 3 → индекс 1.

Бинарный поиск (Binary Search)

Для отсортированных данных. O(log n).

low = 0, high = n-1
Пока low <= high:
    mid = (low + high) // 2
    Если a[mid] == target: Вернуть mid
    Если a[mid] < target: low = mid + 1
    Иначе: high = mid - 1
Вернуть -1

2.3. Алгоритмы на графах

Граф — вершины и рёбра. Обход: DFS (глубина) и BFS (ширина).

# BFS с очередью
queue = [start]
visited = set()
Пока queue:
    node = queue.pop(0)
    visited.add(node)
    Для neighbor в graph[node]:
        Если не visited: queue.append(neighbor)

3. Программирование

Начни с Python — простого языка. Установи Python с python.org. Программы состоят из переменных, циклов, условий и функций.

3.1. Основы

Пример "Hello, World!":

print("Привет, мир!")

3.2. Условные операторы

age = 18
if age >= 18:
    print("Взрослый")
elif age >= 13:
    print("Подросток")
else:
    print("Ребёнок")

3.3. Циклы

for i in range(5):
    print(i)  # 0 1 2 3 4

sum = 0
i = 1
while i <= 10:
    sum += i
    i += 1
print(sum)  # 55

3.4. Функции

def add(a, b):
    return a + b

print(add(3, 5))  # 8

3.5. Списки и словари

# Список
numbers = [1, 2, 3]
numbers.append(4)
print(numbers)  # [1, 2, 3, 4]

# Словарь
person = {"name": "Алекс", "age": 25}
print(person["name"])  # Алекс

3.6. Основы OOP

class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        print("Гав!")

my_dog = Dog("Бобик")
my_dog.bark()
